Streamdata.io SDK

Streamdata.io SDK provides a way to connect to remote server and monitor the connection state. This is achieved with the streamdataio.createEventSource(url), which is the central point of all Streamdata SDK API. The StreamdataEventSource handles connection to the remote server, and provides mechanism to monitor the connection state.

A singleton instance is provided as a "streamdata" global variable.
